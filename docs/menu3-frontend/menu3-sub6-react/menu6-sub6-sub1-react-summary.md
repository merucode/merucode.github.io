## STEP 1. React Basic

### Step 1-1. **`bash`**

```bash
$             # node.js 설치
$ node -v     # node.js 버전 확인 
$ npm -v      # npm 버전 확인

$ npm init react-app .    # 현재 폴더에서 react 프로젝트 시작
$ npm run start           # 개발서버 실행(localhost:3000)


```
