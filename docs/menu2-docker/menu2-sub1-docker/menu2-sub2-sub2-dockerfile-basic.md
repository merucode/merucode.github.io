## STEP . Run

* —no-cache-dir
  * 캐시를 사용하지 않고 pip install을 하라는 뜻. 하드디스크에 공간이 없거나 도커 이미지를 작게 유지하고 싶을 때 사용한다.		

```docker
# install package
COPY requirements.txt requirements.txt
RUN pip install --nocache-dir -r requirements.txt
```
