### 중첩 리스트 합치기

```python
# 중첩 리스트 합치기
my_list = [[7, 6], [5, 4], [1, 2, 3], ['a', 'b']]

sum(my_list, [])    # [7, 6, 5, 4, 1, 2, 3, 'a', 'b']
sum(my_list)        # 오류 발생
```
